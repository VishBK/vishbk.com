Album Art Display

October 2021 - Present

An LED display shows the current album art of the song I'm listening to. If nothing is playing, it shows the date and time. It's powered with an ESP32 microcontroller pulling song information from the Last.FM API. An ambient light sensor is used for auto brightness adjustment and the parts are attached using a 3D printed mount.

  • ESP32
  • 64x64 LED display
  • Ambient light sensor
  • 3D Printer

Software

  • C++
  • PlatformIO
  • Autodesk Fusion
  • Git

Racket State Estimation

January 2023 - May 2024

This was my capstone project for my Robotics Master's at Georgia Tech. I tracked the movement of an opponent's tennis racket (later moving to table tennis for ease of testing) to determine the spin of the ball to better estimate its trajectory. This trajectory is fed into the lab's wheelchair tennis robot to hit the ball back. I used OpenCV and tracked markers on the racket to create bounding boxes, which was trained with Darknet for YOLO. An IMU was used for angular data, and combined with the position data from the YOLO model in a Kalman filter. After testing, I switched to using a more robust human skeleton tracking model for the position data.

Hardware

  • NVIDIA GTX 1070 (Mobile Max-Q)
  • ZED 2 Camera
  • YOST Labs 3-Space Wireless IMU

Software

  • Python
  • C++
  • OpenCV
  • ROS
  • Darknet
  • YOLOv4

Music Data Dashboard

July 2019 - January 2021

A Tableau dashboard displays my music listening history since 2018. The dashboard shows information such as a time heatmap, artist listening streaks, and a year in review summary, similar to Spotify Wrapped. My listens are recorded with an app or a YouTube Music client on my computer and stored in Last.FM. A Python script pulls the Last.FM data using their API and stores it in a CSV to be ingested into Tableau. Track duration data is taken from an unofficial YouTube Music API and combined with the Last.FM data. This script runs nightly on my PC with a scheduled task.

Software

  • Python
  • Tableau
